Highlight locations on a map and thumbnail list

The Story Map Playlist℠ app enables you to feature a selection of locations as clickable map icons and as a list of thumbnails that appear within a side panel. Users can click an item on the thumbnail list or click a map icon to view map pop-ups. Playlists tap a single web map that includes an array of locations, usually derived from a CSV spreadsheet.

View this story map

Examples of Playlist stories

Playlists can be used to highlight a collection of almost any kind of location-specific content. Examples in the gallery include travel destinations, historical photographs, national parks, and golf holes.

Building your story with the Playlist app

Download our source code and refer to the README file for instructions. Playlists are usually derived from spreadsheets with names, descriptions, URLs, and so on. It’s easy to customize titles, text, and tab labels within the Index.html file.
