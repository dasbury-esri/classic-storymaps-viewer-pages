What can you launch?

Map Tour accepts either a 32-character application ID or a 32-character web map ID.
For authored Map Tour stories, appid is the preferred path because it preserves the app's saved narrative and media configuration.

  • App ID example: 2c62acf3468c4cbbba6b82f1035bfe22.
  • Web map example: a5019e8c55d547eab69c0777dcd7509a.
  • Both values must be 32 hexadecimal characters with no spaces, braces or hyphens.
