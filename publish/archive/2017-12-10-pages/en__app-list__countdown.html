Present a ranked list of locations

The Story Map Countdown℠ app enables users to explore a list of places in a particular order. It provides overview and large scale map views simultaneously. While the Countdown app presentation style most effectively shows ranked lists (largest, best, most popular), the app can also be used for regular listings.

Building your story with the Countdown app

As is the case with all of our templates, the Countdown template uses your ArcGIS Online web map as the “back end.”

Because the Countdown template is not currently a hosted template, it is not available through the ArcGIS Online app publishing interface. To create a Countdown story, download the application files and host them yourself.

The download ZIP file contains a README file that specifies what must be included in your web map or web maps. It also explains how to configure the app to point to your web map or web maps.
